# Roadmap de Produto

Prescripta é um motor de apoio à prescrição segura com regras determinísticas,
evidências rastreáveis, revisão humana, IA explicativa e arquitetura preparada
para integração clínica.

## Entregue

- `v0.5.0`: catálogo Brasil/Anvisa, princípio ativo, aliases e vocabulário controlado.
- `v0.6.0`: segurança clínica ampliada e camada de interoperabilidade.
- `v0.7.0`: resumo prático de segurança, perfil funcional e reconciliação granular.
- `v0.7.1`: hardening, configuração de IA via UI, qualidade textual e fechamento de lacunas.
- `v0.8.0`: relatórios, exportação e auditoria avançada.
- `v0.8.1`: prontidão de produto, UX, IA hardening, docs, assets e amarração clínica.
- `v0.8.2`: protocolos rápidos/emergência demonstrativos, frontend polish, README forte,
  docs de onboarding e assets públicos ampliados.
- `v0.8.3`: inteligencia clinica assistida, hist?rico longitudinal do paciente,
  protocolos integrados a `GeneratedReport`, catalogo farmacol?gico ampliavel,
  prompts por modulo e frontend com visao clinica/tecnica.

## Próximas Versões

- `v0.9.0`: Docker/PostgreSQL/migrações/deploy.
- `v1.0.0`: versão final de portfólio.

## Limites

- Não é dispositivo médico validado.
- Não substitui avaliação profissional.
- Não implementa integração hospitalar real sem API oficial, contrato, segurança e LGPD.
- Protocolos rápidos seguem educacionais, demonstrativos e dependentes de julgamento humano.
