# Visão de Produto

Prescripta demonstra um fluxo educacional de apoio à revisão de prescrição.

## Público

- Estudantes e profissionais avaliando arquitetura de healthtech.
- Recrutadores e avaliadores técnicos em portfólio.
- Desenvolvedores estudando separação entre UI, API e regra de domínio.

## Proposta

Criar uma experiência objetiva para cadastrar pacientes, cadastrar medicamentos, simular prescrições e visualizar alertas auditáveis.

## Não Objetivos

- Não substituir julgamento profissional.
- Não operar como dispositivo médico.
- Não usar dados reais.
- Não delegar decisões críticas a IA.
