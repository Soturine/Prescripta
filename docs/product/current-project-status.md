# Estado atual do projeto

O Prescripta v0.8.5 é um portfólio healthtech demonstrativo executável com API, interface, banco
SQLite local, regras determinísticas, IA assistiva opcional, relatórios e auditoria. Não é produto
clínico validado nem pronto para operação hospitalar.

Funcionalmente estão disponíveis pacientes, perfil funcional, documentos textuais, catálogo,
checagem, Dose Intelligence, sinais psicotrópicos, policy demo, protocolos, importações,
reconciliação, relatórios, IA/fallback, usuários e auditoria. A matriz de aceite distingue o que está
concluído, parcial, bloqueado por validação externa e adiado.
