# Notas do Seed de Medicamentos

O seed é fictício e educacional. Ele existe para facilitar demonstrações do fluxo.

## Medicamentos

- `Ibuvida`: exemplo com ibuprofeno, dose máxima e contraindicação renal.
- `Clarimicina`: exemplo de macrolídeo para interação com sinvastatina.
- `Metfor`: exemplo de metformina com contraindicação renal.
- `Sertral`: exemplo de sertralina para interação demonstrativa.

## Interações Demonstrativas

- Varfarina + ibuprofeno.
- Enalapril + espironolactona.
- Sinvastatina + claritromicina.
- Metformina + contraste iodado.
- Sertralina + tramadol.

Essas interações não são uma base clínica completa.
