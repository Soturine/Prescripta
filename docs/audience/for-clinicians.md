# Prescripta Para Medicos

Esta visao resume como usar o Prescripta em fluxo clínico demonstrativo.

## Fluxo Recomendado

1. Abra o paciente e confira alergias, medicamentos atuais, comorbidades e
   histórico longitudinal.
2. Revise laudos/documentos pendentes antes de confiar no bundle clínico.
3. Cheque a prescrição com dose, frequencia, via e duracao.
4. Leia a decisão resumida, alertas praticos, dados considerados e dados
   faltantes.
5. Gere orientação ao paciente ou relatório quando precisar documentar o caso.
6. Use Protocolos para cenarios rapidos, sempre com julgamento profissional.

## Como Ler Alertas

Alertas indicam necessidade de revisão, não diagnostico nem conduta automatica.
Risco alto ou critico exige verificacao de fonte, contexto clínico e protocolo
institucional.

## Quando Abrir Detalhes Tecnicos

Abra detalhes tecnicos quando precisar ver regras disparadas, evidencia, bundle,
RAG, hash, provider/modelo de IA ou payload exportavel.
